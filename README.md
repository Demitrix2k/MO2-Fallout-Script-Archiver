A PowerShell-based automation tool for Mod Organizer 2 (MO2) to consolidate multiple small mods into organized BSA archives, improve performance, and simplify mod management. See `Script-Archiver-windows/` and `Script-Archiver-linux/` for platform-specific instructions.

## Project Structure

- `Script-Archiver-windows/` – Windows launcher, scripts, and BSArch
- `Script-Archiver-linux/` – Linux launcher, scripts, and BSArch (runs via Wine)
